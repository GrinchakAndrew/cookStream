Screen 4:

  • Screens: UI # 4: Stream screen
    • User-story:
      When: user clicks on the Stream icon on the previous UI #3
      Then: user is redirected to see the video stream
      And: I can click-select the categories 'Soups' | 'Grills', etc.

    • User-story:
      When: user clicks on a category
      Then: its anchor becomes blueish in color
      And: filtering of the video streams takes place implicitly to show only the matched videos by category
      And: can swipe upwards to see the next video feed in the stream

    • User-story: video data
      When: User is on the video in the stream
      Then: Each video streaming video has the following data:
      (1) Cooking time minutes and seconds
      (2) Number of the calories per dish
      (3) Elements number (i.e., ingerdients)

    • User-story: Footer
      Footer has icons to redirect to each of the 3 categories: 'Stream', 'Fitness', 'Cook Now'


    • User-story: Unfiltered Feed View
      When: User is on the video feed
      Then: User should be able to view the latest videos up top
      And: User should be able to default to unfiltered view of the video feed before using filter settings

    • User-story: Filter Reset
      When: User turns off the filtering options by pressing the X on the filter
      Then: User should have an action sheet or a popup option to confirm filter reset

    • User-story: All Categories
      When: User taps on All Categories
      Then: User should view videos in the stream by all categories
