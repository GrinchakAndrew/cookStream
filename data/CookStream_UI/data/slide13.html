    Dietary Preferences:
    • User-story: Dietary Preferences
      Description : User sets his Dietary Preference, which influence his Cookstream Video Feed
      When : User checks his Dietary Preferences Then : User should see the relevant news feed videos in the Cookstream section
      And : User should be able to change his Dietary Preferences at any time to update Cookstream videos in the feed

    • User-story: Dietary Preferences conflicting with manual input when searching for videos by ingerdients
      When : User has toggled off Meat category in the Dietary Preferences
      And : Goes to searchbar to locate videos by Meat category and\or any of its child ingeredients
      Then : The manual input should overpower the Dietary Preferences settigns
      And : Provide the search results by manual input

    • User-story: Dietary Preferences conflicting with manual input defaulting to settings in the Dietary Preferences
      When : User has toggled off 'Meat' category in the Dietary Preferences
      And : Goes to his general cook stream to see the videos selection
      Then : The video selection should default to Dietary Preferences settigns
      And : Show the corresponding results
