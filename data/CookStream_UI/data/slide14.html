    Personal Data Changes:
    • User-story: Personal Data Changes
      Description : User can have multiple phone numbers linked to his account, and therefore can change his phone number at any time
      When : User changes his Photo Then : User should be able to take a photo
      And : User should be able to select any existing photo from his Gallery
      And : User should be able to crop and resize his image

    • User-story: Changed Phone Update SMS-Verification-1
      Description : User inputted his new phone number into the Phone Number input field:
      When : User is done with input of the phone number, and presses the DONE button on the keyboard
      And : server-side receives API request for verification,
      While : the server is in process of the response, the number is blocked and cannot be changed
      And : user is shown a popup -> "your verification code will be sent shortly, pls., check it"

    • User-story: Changed Phone Update SMS-Verification-2
      When : user receives the SMS-code
      Then : user inputs the SMS-code
      And : user presses the button 'Save'
      And : App dispatches the information to the server
      And : spinner is shown to the user while the server verifies the SMS code with the inputted sms -code and updates the data

    • User-story: Changed Phone Update SMS-Verification-3
      When : server responds with 'SUCCESS' status of the request
      Then : the number of the telephone which is updated is marked with the V-shaped Tick
      in the upper right corner above the telephone number input
