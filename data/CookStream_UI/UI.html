  • UI 1
    • Epic: Sign-up flow
      As: a User, when I am on the UI #1
      I want: to see the logo of the application
      And: The name of the application
      And: be able to sign up via:
      (1) FB account or with my email
      (2) through 'Already have an account' option

    • User-story: Sign-up | Sign-in via phone-number:
      When: I input my phone-number Then: I can validate the number via received sms
      And: And input my security code

    • User-story: Sign-up | Sign-in FB account:
      When: I on on UI # 1 And: I want to sign up | sign-in via my FB account
      Then: I can do so via login to my FB account in the application

    • User-story: Sign-up | Sign-in via telephone number:
      When: user inputs his telephone number And: And user inputs his password Then: he can register in the app to create account

    • User-story: When: user clicks on 'Already Have An Account' And: received the sms with the pass code Then: And user can sign in to the application via input of the received security code from the sms

    • User-story: When:user has deleted the application from his device And: user has re-installed the app again Then: User will see the sign up screen to input the phone-number and receive the sms with the security code
